# Project Context: Gesture Music App
- **Tech Stack**: React, Vite 8 (Beta), Framer Motion.
- **Developer Background**: Master's student in Web Engineering at TU Chemnitz.
- **Coding Standards**: 
  - Prioritize modularity and scalable architecture.
  - Follow React best practices (custom hooks, functional components).
  - Optimize for performance and Core Web Vitals.
- **Project Goal**: A web-based music controller that uses gestures (swipes, and eventually webcam-based hand tracking) to play and skip music.
